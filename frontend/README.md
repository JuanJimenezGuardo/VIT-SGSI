# Frontend VIT (React + Vite)

Aplicacion SPA para visualizacion y gestion operativa de modulos SGSI.

## Tecnologias

- React 18
- Vite
- Axios para consumo de API

## Ejecucion

```bash
npm install
npm run dev
```

Servidor local por defecto: http://localhost:3000

## Build de Produccion

```bash
npm run build
npm run preview
```

## Alcance Funcional en Esta Version

- autenticacion y rutas protegidas
- vistas de proyectos y detalle
- integracion con endpoints backend
- componentes de soporte para pruebas funcionales
